1. LangChain (랭체인)

  정의: LLM(대규모 언어 모델)을 이용한 애플리케이션 개발을 쉽고 빠르게 도와주는 오픈소스 프레임워크입니다.
  핵심 역할: 모델(LLM), 데이터 소스, 외부 도구(API 등)를 서로 연결(Chain)하는 역할을 합니다.
  특징: 선형적인 흐름(A → B → C)을 구성하는 데 최적화되어 있으며, 템플릿 관리, 데이터 검색(RAG), 메모리 관리 기능을 제공합니다.

2. LangGraph (랭그래프)

  정의: LangChain 위에 구축된 라이브러리로, LLM을 이용해 순환(Circular) 및 상태 유지(Stateful) 워크플로우를 만들 수 있게 해주는 프레임워크입니다.
  핵심 역할: 단순한 일직선 구조가 아닌, 조건에 따라 다시 이전 단계로 돌아가거나 반복하는 그래프(Graph) 구조를 구현합니다.
  특징: '상태(State)'를 공유하므로 복잡한 에이전트(Agent) 시스템을 만들 때 필수적입니다.

3. Superstep (슈퍼스탭)

  정의: LangGraph 내에서 발생하는 하나의 논리적 실행 단위 또는 그래프의 한 단계 이동을 의미합니다.
  작동 원리: LangGraph는 'Pregel'이라는 분산 그래프 처리 시스템의 개념을 차용합니다.
  현재 활성화된 노드들이 작업을 수행합니다.
  노드들이 상태(State)를 업데이트합니다.
  다음 노드로 신호를 보냅니다.

  해설: 이 한 번의 주기(Step)를 Superstep이라고 부릅니다. 그래프 실행 시 step 단위로 로그를 확인하거나 루프가 무한히 돌지 않도록 recursion_limit을 설정할 때 이 단위를 기준으로 제어합니다.
